# Лабораторная работа №4. Работа с Markdown-разметкой
***

**Markdown** — это *~~лёгкий~~* язык разметки, который используется для оформления
README-файлов, документации, отчётов и даже научных текстов. Он прост в
использовании, хорошо читается как в сыром виде, так и после рендеринга, и широко
поддерживается GitHub, VS Code и многими платформами.
**LaTeX** — мощная система вёрстки, которая позволяет оформлять формулы,
математические выражения и структурированные документы прямо внутри Markdownфайлов (например, в GitHub или Obsidian).
>[!WARNING] очень много информации
